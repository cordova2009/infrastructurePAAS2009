###银行卡信息查询接口



1）请求说明

http请求方式: post

    http://IP:PORT/if/outsource/getbankcardinfo


POST数据格式：JSON

    {
        "userid":"123",
    }


参数|必须|说明
------|------|-------
userid|是|用户id


2）返回说明

正常时的返回JSON数据包示例：

    {
        "errcode":0,"errmsg":"信息查询成功",
        "data":[{
            "account":"麦圈深圳",
            "cardNum":"13912345678", 
            "bankaddress":"支行"
        }]
    }

错误时的JSON数据包示例：

    {
        "errcode":10000,"errmsg":"获取数据失败"
    }


参数|必须|说明
----|----|----
account|是|银行账户名
cardNum|是|银行账号
bankaddress|是|银行开户地址

###查询发包方/承包方列表

    http://IP:PORT/if/outsource/getcreditlist

        "seach":{
            "companyname":"深圳沈健",
            "legalname":"李四",
            "realname":"李小四",
            "mobilenum":"13692228033"
        }
    }


参数|必须|说明
------|------|-------
userid|是|用户id
companyname|否|公司名称查询 
legalname|否|法人代表名查询
realname|否|用户名查询
mobilenum|否|用户关联手机号查询


2）返回说明

    {
        "errcode":0,"errmsg":"获取成功",
        "data":[{
            "id":"01",
            "companyname":"深发展",
            "legalname":"",
            "level":"1",
            "realname":"",
            "projectCategory":"土石方",
            "mobilenum":"1369222930"
            }]
    }

错误时的JSON数据包示例：

    {
        "errcode":10000,"errmsg":"获取失败"
    }

参数|必须|说明
------|------|-------
id|是|流水id
companyname|是|公司名称查询 
legalname|是|法人代表名查询
realname|是|用户名查询
mobilenum|是|用户关联手机号查询
level|是|等级
projectCategory|是|工程类别
