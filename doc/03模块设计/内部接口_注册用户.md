###禁用启用


1）请求说明

http请求方式: post

    http://IP:PORT/if/setStatus


POST数据格式：JSON

    {
        "userid":"123",
        "status":"Y"
    }


参数|必须|说明
------|------|-------
userid|是|用户id
status|是|状态 Y启用  N禁用

2）返回说明

正常时的返回JSON数据包示例：

    {
        "errcode":0,"errmsg":"设置成功"
    }

错误时的JSON数据包示例：

    {
        "errcode":10000,"errmsg":"设置失败"
    }

###查看评价信息



1）请求说明

http请求方式: post

    http://IP:PORT/if/getreview


POST数据格式：JSON

    {
        "userid":"222",
        "token":"sdf23"
    }


参数|必须|说明
------|------|-------
userid|是|用户id
token|是|登录用户的token


2）返回说明

正常时的返回JSON数据包示例：

    {
        "errcode":0,"errmsg":"信息查询成功",
        "contract":{
            "reviewname":"评论人名",
            "reviewid":"评论人id", 
            "username":"被评论人名",
            "reviewtime":"评论时间",
            "reviewcontent":"评论内容",
            "reviewpoint":"评价得分"
        }
    }

错误时的JSON数据包示例：

    {
        "errcode":10000,"errmsg":"获取数据失败"
    }


参数|必须|说明
----|----|----
reviewname|是|评论人名
reviewid|是|评论人id
username|是|被评论人
reviewtime|是|评论时间
reviewcontent|是|评论内容
reviewpoint|是|评价分


###查看修改记录


1）请求说明

http请求方式: post

    http://IP:PORT/if/geteditrecode


POST数据格式：JSON

    {
        "userid":"222",
        "token":"se231sd",
        "pageindex":"10",
        "pagesize":,"100"
    }


参数|必须|说明
------|------|-------
userid|是|用户id
token|是|登录用户的token
pageindex|否|分页页码



2）返回说明

正常时的返回JSON数据包示例：

    {
        "errcode":0,"errmsg":"信息查询成功",
        "data":[{
            "id":"记录id",
            "type":"工程类", 
            "field":"user",
            "status":"OK"
        }]
        ,"pageindex":"1"
        ,"pagesize":"20"
        ,"allsize":"200"
    }

错误时的JSON数据包示例：

    {
        "errcode":10000,"errmsg":"获取数据失败"
    }


参数|必须|说明
----|----|----
id|是|修改记录id
type|是|修改类别
status|是|状态 OK成功 ,FAIL失败
field|是|修改字段名
pageindex|否|分页页码
pagesize|否|分页大小
allsize|否|总记录数

    {
        "userid":"222",

        "realname":"王四",
        "newpwd":"23123",
        "cardID":"6123392783879498"
    }


参数|必须|说明
------|------|-------
userid|是|用户id
