#目的

该文档定义用户消息模块接口，并指导开发。

#接口

##向用户发送站内消息接口

向用户发送站内消息,主要是系统内部执行.

1）请求说明

http请求方式: post

    http://IP:PORT/if/userMsg/addMsg


POST数据格式：JSON

    {
        "app":{
            "appId":"zjhtwallet",
            "timeStamp":"TIMESTAMP", 
            "nonce":"NONCE",
            "signature":"SIGNATURE"
        },
	    "body":{
            "userId":"erwd123",
            "msgContent":"恭喜您在招标\"天宫一号\"项目中中标了",
            "msgType":"PRI",
            "msgTitle":"中标通知"
        }
    }


参数|必须|说明
------|------|-------
userId|是|接收通知的用户
msgTitle|是|标题
msgContent|是|内容
msgType|是|通知内容,PRI个人,OTH 其它



2）返回说明

正常时的返回JSON数据包示例：

    {
        "errcode":0,"errmsg":"添加用户消息成功"
        
    }

错误时的JSON数据包示例：

    {
        "errcode":10000,"errmsg":"添加用户消息失败"
    }
