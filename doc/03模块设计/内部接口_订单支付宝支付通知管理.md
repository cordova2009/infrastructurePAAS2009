###支付通知接口

1）请求说明

http请求方式：POST

    http://IP:80/if/order/notify    

POST数据格式：JSON

POST数据例子：


    {
        "errcode":0, "errmsg":"支付成功","orderId":"FN2010219392838232","payTime":"2015-02-02 12:12:12"
           
    }

参数|是否必须|说明
---|-------|----
orderId|是|订单id
errcode|是|支付结果，0-成功，1-失败
payTime|是|支付时间(字符串格式) yyyy-MM-dd HH:mm:ss
errmsg|是|支付信息
