#接口


###招标方资质认证接口

前端页面通过该接口认证招标方信息


http请求方式: post

前置条件：用户必须先登录

    http://ip:port/if/userInfo/authenticationPrivilege


POST数据格式：JSON

    {
        "app":{
            "appId":"zjhtwallet",
            "timeStamp":"TIMESTAMP", 
            "nonce":"NONCE",
            "signature":"SIGNATURE"
        },        
        "body":
        {
	    "userId":2,	
	    "longName":"深圳蜂鸟娱乐技术有限公司",
	    "shortName":"蜂鸟娱乐",
	    "regTime":"2014-04-05",
	    "operationPeriod":"公司营业期限",
	    "registeredCapital":500000,
	    "address":"",
	    "email":"",
	    "businessScope":"经营范围",
	    "synopsis":"公司简介",
	    "projectCategoryId":[1,3,5],//工程类别
	    "businessLicenseNum":"36544789",
	    "businessLicenseUrl":"营业执照上传地址",
	    "taxRegistrationNum":"2434567",
	    "taxRegistrationUrl":"税务证书上传地址",
	    "organizationCodeNum":"3w3454676",
	    "organizationCodeUrl":"组织机构代码证上传地址",
	    "contactsName":"联系人",
	    "contactsPhone":"1234567",
	    "legalPersonName":"张三",
	    "legalPersonIdentityCardName":"420923199205049230121"
            "legalPersonIdentityCardUrl":"",
	    "legalPersonAuthorizationUrl":"",
	    "enterpriseQualification":[{"certificateId":2,"certificateUrl":""},{"certificateId":3,"certificateUrl":""}],
	    "bank":"招商银行",
	    "subBranch":"支行",
	    "bankCardNum":"",
	    "accountName":"账户名称",

        }
    } 


参数|是否必须|说明
----|----|-----
appId|是|应用ID
timestamp|是|时间戳
nonce|是|随机数
signature|是|签名值,MD5(按值的字典顺序排列组合成字符串(appId,appKey,app.nonce,app.timeStamp))
userId|是|用户Id
longName|是|公司全称
shortName|是|公司简称
regTime|是|注册时间
operationPeriod|是|公司营业期限
registeredCapital|是|注册资本
address|是|公司地址
email|是|公司邮箱
businessScope|是|经营范围
synopsis|是|公司简介
projectCategoryId|是|工程类别
businessLicenseNum|是|营业执照号
businessLicenseUrl|是|营业执照上传地址
taxRegistrationNum|是|税务证书编号
taxRegistrationUrl|是|税务证书上传地址
organizationCodeNum|是|组织机构代码
organizationCodeUrl|是|组织机构代码证上传地址
contactsName|是|联系人姓名
contactsPhone|是|联系人手机号
legalPersonName|是|法人姓名
legalPersonIdentityCardName|是|法人身份证号
legalPersonIdentityCardUrl|是|法人身份证上传地址
legalPersonAuthorizationUrl|否|法人授权书上传地址
enterpriseQualification.certificateId|是|资质证书Id
enterpriseQualification.certificateUrl|是|资质证书上传地址
bank|是|开户银行
subBranch|是|开户支行
bankCardNum|是|银行账户
accountName|是|账户名称

2）返回说明

正常时的返回JSON数据包示例：
 
    {"errcode":0,"errmsg":"招标方认证信息保存成功"
    }

错误时的JSON数据包示例：

    {"errcode":10000,"errmsg":"招标方认证信息保存失败"}

###会员信息查询接口

前端页面通过该接口查询会员接口

http请求方式: post


    http://ip:port/if/userInfo/queryMemberInfo


POST数据格式：JSON

    {
        "app":{
            "appId":"zjhtwallet",
            "timeStamp":"TIMESTAMP", 
            "nonce":"NONCE",
            "signature":"SIGNATURE"
        },        
        "body":
        {
	    "type":"tender",
	    "userId":2
	 }
    }

2）返回说明

正常时的返回JSON数据包示例：
 
    {"errcode":0,"errmsg":"查询会员信息成功",
    "isMember":true,
    "memberStartTime":"",
    "memberEndTime":"",
    "memberType":"",
    "memberContent":""
    }

错误时的JSON数据包示例：

    {"errcode":10000,"errmsg":"查询会员信息失败"}

参数|是否必须|说明
----|----|-----
isMember|是|是否为会员
memberStartTime|否|会员开始时间
memberEndTime|否|会员结束时间
memberType|否|会员类型
memberContent|否|会员特权


###成为招标方会员接口

前端页面通过该接口成为招标方会员接口


http请求方式: post

前置条件：用户必须先登录,


    http://ip:port/if/userInfo/becomePrivilegeMember


POST数据格式：JSON

    {
        "app":{
            "appId":"zjhtwallet",
            "timeStamp":"TIMESTAMP", 
            "nonce":"NONCE",
            "signature":"SIGNATURE"
        },        
        "body":
        {
	    "userId":2,
	    "memberType":"",
	    "memberDuration":3,
	    "payMethod":"",
	    "payAmount":50000

	 }
    }

参数|是否必须|说明
----|----|-----
userId|是|用户Id
memberType|是|会员类型
memberDuration|是|购买会员期限
payMethod|是|支付方式
memberContent|是|支付金额


2）返回说明

正常时的返回JSON数据包示例：
 
    {"errcode":0,"errmsg":"购买会员成功"}

错误时的JSON数据包示例：

    {"errcode":10000,"errmsg":"购买会员失败"}
